# 📝 React Todo App

A simple and responsive **Todo List Application** built with React.

Users can add, complete, and delete tasks to manage daily activities efficiently.

✔ Add new tasks  
✔ Mark tasks as completed  
✔ Delete tasks  
✔ Instant UI updates  
✔ Responsive design
